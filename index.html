SteamZL Loader Guide

üìñ What is a Manifest File?

Manifest files (.acf) are Steam AppManifest files that tell Steam about installed games. They contain the App ID, installation directory, and other metadata. These files are essential for Steam to recognize and manage game installations.

üîß What is a Lua File?

Lua files are script files that execute code to add app IDs to your Steam instance. They work with SteamTools and other Steam emulators to load games and DLC. When executed, these scripts register applications with the emulator.

üì• Installation Steps

  1. Generate your manifest files using SteamZL (use any of the generator tabs above)
  2. Copy .acf files to your Steam/steamapps folder
  3. Copy .lua files to your loader directory (typically in your Steam emulator folder)
  4. Restart Steam or your loader application
  5. Games should now appear in your library

‚ö° Activating the Loader

The loader activates automatically when placed in the correct directory. Most Steam emulators will execute .lua files on startup. If your emulator has a scripts or plugins folder, place the loader.lua file there. The loader will initialize when you launch the emulator.

‚ùå Games not appearing:

‚ùå DLC not loading:

üìÑ File Format Explanation

Manifest File (.acf) Structure:

"AppState" { "AppID" "730" // Game's unique identifier "Universe" "1" // Steam universe (always 1) "installdir" "game_730" // Installation directory name "StateFlags" "1026" // Installation state flags }

Lua File Structure:

addappid(730, 1, "K_Game_730") // Parameters: // - 730: App ID // - 1: Universe ID // - "K_Game_730": Internal identifier key

How It Works

What are App IDs?

App IDs are unique identifiers for games and applications on Steam. Each game has its own App ID that Steam uses to identify it.

How to find App IDs:

  • Visit SteamDB.info
  • Search for your game
  • The App ID is displayed on the game's page

Using the generated files:

The generated .acf manifest files and .lua files can be used with Steamtools to add online functionality to your games. Place the manifest files in your Steam userdata folder and the lua files in the appropriate directory.
